---
name: Guides and instructions location
description: Where to store guides, instructions, and skills in the project
type: feedback
---

When creating guides, instructions, or skills, place them in `.claude/<file_type>/file.md` at the root of the project.

**Why:** Keeps these resources organized and discoverable in a dedicated `.claude/` directory at the project root.

**How to apply:** Any time you create new guides, instructions, skills, or similar documentation, use the `.claude/<file_type>/` directory structure rather than storing them elsewhere.
